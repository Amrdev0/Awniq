# Code of Conduct

## Our Standard

Awniq welcomes contributors who are respectful, focused, and constructive. This project serves aid operations, where trust and care matter. We expect everyone to:

- Communicate respectfully.
- Assume good intent while still being precise about technical concerns.
- Keep discussions focused on the issue or pull request.
- Respect privacy and avoid sharing real beneficiary, donor, or operational data.
- Accept maintainer decisions when a feature is outside the current scope.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without explicit permission.
- Introducing or requesting unsafe handling of sensitive data.
- Repeatedly derailing technical discussions.
- Misrepresenting demo data as real organizational data.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; reject pull requests; or restrict participation when behavior harms the project or community.

If you need to report a conduct issue, contact the maintainers privately. Include links, screenshots, timestamps, and any other context that helps review the situation fairly.
