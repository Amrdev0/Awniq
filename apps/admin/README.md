# Awniq Admin

React TypeScript admin frontend for Awniq.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Environment:

```txt
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

Default URL:

```txt
http://127.0.0.1:5173
```

Public portal route:

```txt
http://127.0.0.1:5173/public
```

## Commands

```bash
npm run lint
npm run test
npm run build
```
